# 恋上链

基于区块链的爱情日记，能够记录情侣之间的互动信息。

也可以记录自己的生活和感悟。

[恋上链Demo(非实际版本)](http://ustczf.com)

## Timeline系统

每个人都有自己的一条时间线，可以在上面记录，一旦记录就无法删除。

## 前端

Vue + Bootstrap

## 后端

目前可以使用以太坊和公信宝（正在调试中），正在开发通过MetaMask的接口，可以在浏览器上直接把信息上链。
